# Practice Problems - Simple Blog

This directory contains practice problems to help you master multi-page applications and navigation.

## Running the Tests

1. Install dependencies:
   ```bash
   pip install -r ../../../requirements.txt
   ```

2. Run tests for a specific problem:
   ```bash
   pytest test_problem1.py -v
   pytest test_problem2.py -v
   pytest test_problem3.py -v
   ```

3. Run all tests:
   ```bash
   pytest -v
   ```

## Problems

### Problem 1: Basic Blog with Posts
**Difficulty:** Medium (40-50 minutes)
Create a multi-page blog with navigation and template inheritance.

### Problem 2: Blog with Categories
**Difficulty:** Medium (40-50 minutes)
Add category filtering and CSS Grid layouts.

### Problem 3: Responsive Blog Design
**Difficulty:** Hard (50-60 minutes)
Implement responsive design and professional styling.

## Tips

- Use template inheritance to avoid repeating HTML
- Organize your static files (CSS, JS, images) properly
- Test responsive design by resizing your browser
- Use CSS Grid and Flexbox for modern layouts
- Always include viewport meta tag for mobile
