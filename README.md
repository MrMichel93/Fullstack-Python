# Full-Stack Development with Python

**A Project-Based Course for Ages 16‚Äì20**

Learn full-stack web development by building real projects. This course focuses on practical skills, not theory. You'll build 4 portfolio-ready applications using Python, Flask, HTML, CSS, and SQLite.

## Prerequisites
- Basic Python knowledge (variables, functions, loops)
- Basic HTML knowledge (tags, forms)
- No prior web development experience required

## Tech Stack
- **Backend**: Flask
- **Frontend**: HTML, CSS, minimal JavaScript
- **Database**: SQLite
- **Python**: 3.8+

## Getting Started

1. Clone this repository
2. Install Python 3.8 or higher
3. Start with the [Flask Setup Guide](./modules/00-flask-setup/)
4. Read the [Architecture Primer](./modules/01-architecture-primer/)
5. Work through projects in order
6. Complete TODO labs in each project
7. Try extension challenges

## Installation

```bash
# Clone the repository
git clone https://github.com/MrMichel93/Fullstack-Python.git
cd Fullstack-Python

# Install dependencies (from any project directory)
pip install -r requirements.txt
```
