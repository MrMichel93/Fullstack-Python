## Tech Stack
- **Backend**: Flask or FastAPI
- **Frontend**: HTML, CSS, minimal JavaScript
- **Database**: SQLite
- **Python**: 3.8+

## Getting Started

1. Clone this repository
2. Install Python 3.8 or higher
3. Start with the Architecture Primer
4. Work through projects in order
5. Complete TODO labs in each project
6. Try extension challenges

## Project Structure

Each project includes:
- `README.md` - Project overview and learning objectives
- `starter/` - Starting code with TODOs
- `solution/` - Complete working solution
- `challenges.md` - Extension ideas to practice

## Installation

```bash
# Clone the repository
git clone https://github.com/MrMichel93/Fullstack-Python.git
cd Fullstack-Python

# Install dependencies (from any project directory)
pip install -r requirements.txt
```
